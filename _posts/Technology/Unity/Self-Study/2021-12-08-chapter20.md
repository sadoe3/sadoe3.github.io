## How to close a game in Unity
- Unity에서 game을 종료시킬 때에는 editor version에서 종료시키는 것과 build version에서 종료시키는 것들을 모두 구현해야한다.
- 다음과 같은 코드를 통해 종료시키면 된다.
- 첫번째 방법
```c#
using UnityEngine;
#if UNITY_EDITOR
using UnityEditor;
#endif
... // some codes
    #if UNITY_EDITOR
    EditorApplication.isPlaying = false;
    #else
    Application.Quit();
    #endif
... // some codes
```
